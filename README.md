# Pricing Plan Selector

Projeto desenvolvido com HTML e CSS que apresenta um componente de seleção de planos utilizando radio buttons personalizados. O usuário pode escolher entre diferentes opções através de uma interface moderna e amigável.

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3

## ✨ Funcionalidades

- Seleção de planos
- Radio buttons personalizados
- Feedback visual instantâneo
- Efeito hover
- Design moderno
- Sem JavaScript
- Interface intuitiva

## 📂 Estrutura do Projeto

```bash
Pricing-Plan-Selector/
│
├── index.html
├── css/
│   └── style.css
└── README.md
```

## 🎯 Como Executar

1. Clone ou baixe o projeto.
2. Abra a pasta do projeto.
3. Execute o arquivo `index.html` no navegador.

Praticar a criação de componentes de seleção modernos para formulários, sistemas de assinatura, planos de serviços e interfaces de checkout.
